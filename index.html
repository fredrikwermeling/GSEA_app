How to use Enrich

Enrich runs Gene Set Enrichment Analysis (GSEA) entirely in your browser. No data is uploaded to any server.

1. Prepare your data

Your input file should be a .csv or .tsv with at least two columns:

  • Gene name column — standard gene symbols (e.g. TP53, BRCA1)
  • Ranking metric column — a numeric value to rank genes (e.g. log2 fold change, t-statistic, or signal-to-noise ratio)

Save your spreadsheet as CSV (comma-separated) or TSV (tab-separated) from Excel, Google Sheets, or R. Ensure the first row contains column headers.

2. Upload and configure

  1. Click Choose file or use Load Example Data to try with sample data
  2. Select which column contains gene names and which contains the ranking metric
  3. Genes will be automatically sorted from highest to lowest metric value

3. Choose gene set collections

Select one or more MSigDB collections (Hallmark is recommended to start). You can also upload a custom GMT file.

4. Run the analysis

Click Run GSEA. The analysis runs in your browser using a Web Worker. With Hallmark (50 sets) this takes seconds; larger collections take longer.

5. Interpret the results

  • Overview — Lollipop plot showing top enriched gene sets sorted by NES
  • Enrichment Plot — Classic GSEA enrichment score curve for individual gene sets
  • Overlap — Jaccard similarity heatmap of significant gene sets
  • Results Table — Full sortable/filterable table with CSV export

Key statistics

  • NES (Normalized Enrichment Score) — positive = upregulated, negative = downregulated
  • FDR (False Discovery Rate) — <0.25 is suggestive, <0.05 is significant
  • Leading Edge — the subset of genes driving the enrichment signal

Gene set collection strategy

MSigDB contains over 33,000 gene sets across 7 collections. Running all at once is possible but takes several minutes and produces strict FDR corrections. A recommended approach:

  1. Start with Hallmark (50 sets) — well-defined biological pathways, fast and highly interpretable
  2. Explore specific collections — if Hallmark shows inflammatory signals, try C7 (Immunologic); for metabolic hits, try C2 (Curated: KEGG/Reactome)
  3. Use the Gene Set Browser — search for specific pathways or genes of interest across all collections
  4. Iterative permutation strategy — for large analyses, you can run with lower permutations (e.g. 200) for faster screening, then filter the results table and use “Re-run filtered” with ≥1000 permutations. Note: low permutations give coarser p-values, so borderline hits (FDR ~0.2–0.3) may be missed in the initial screen
  5. Use the Data Type selector — set to "CRISPR / Perturbation" if your data is from a genetic screen or perturbation experiment, for correct interpretation

How to Interpret Enrichment Data

What is GSEA?

Gene Set Enrichment Analysis (GSEA) tests whether a predefined set of genes shows statistically significant, concordant differences between two biological states. Rather than looking at individual genes, GSEA asks: are the genes in this pathway collectively shifted toward one end of my ranked list?

The Enrichment Score (ES) Curve

The algorithm walks down your ranked gene list from top to bottom. At each position:

  • If the gene is in the gene set → the running score steps up (proportional to the gene's ranking metric)
  • If the gene is not in the gene set → the running score steps down (by a small constant)

The Enrichment Score (ES) is the maximum deviation from zero. The genes before this peak are the leading edge — the core genes driving the enrichment signal.

Enrichment Score curve — schematic
  ES
   ^
   |        .---.
   |       /     \        ← Peak = ES (enrichment score)
   |      /       \
   |     /   Leading
   |    /    edge
   |   /           \
   +--/-------------\---------> Rank
   |                 \       /
   |                  \.___./
   |
   |  ||||  | || |           |    ← Hit markers (genes in set)
   +---------------------------------->
   |  High metric          Low metric

If genes are perfectly evenly spread across the ranked list (no enrichment), the ES curve stays near zero — a flat line. The more the gene set clusters at one end, the larger the peak.

Understanding the Statistics

Statistic What it means
ES Raw enrichment score — the maximum deviation from zero. Positive = enriched at the top of the list; negative = enriched at the bottom.
NES Normalized Enrichment Score — ES normalized for gene set size, allowing comparison across different gene sets.
p-value Empirical p-value from permutation testing. How often a random gene set of the same size achieves an ES this extreme.
FDR False Discovery Rate (Benjamini-Hochberg). The estimated proportion of false positives among all gene sets with this FDR or better. FDR < 0.25 is the standard threshold; FDR < 0.05 indicates strong evidence.

Example 1: Expression Data

Scenario: Proteins are quantified after stimulating cells with a ligand. Genes/proteins are ranked from high (most upregulated, stim vs ctrl) to low (most downregulated).

  • Positive NES (peak on the left) → This pathway is upregulated by the stimulation. The leading edge genes are the most upregulated members of the pathway.
  • Negative NES (peak on the right) → This pathway is downregulated by the stimulation.

Example 2: CRISPR / Perturbation Screen

Scenario: A CRISPR screen tests how gene knockouts affect cell survival under drug treatment. Genes are ranked from high (sgRNA/KO enriched in +drug vs ctrl — KO makes cells more resistant) to low (sgRNA/KO depleted — KO makes cells more sensitive).

  • Positive NES → Knocking out genes in this pathway makes cells more resistant to the drug (enriched/positively selected).
  • Negative NES → Knocking out genes in this pathway makes cells more sensitive to the drug (depleted/negatively selected).

CRISPR screens can also test other phenotypes (e.g., sorting cells by a surface marker, growth vs. no growth). The interpretation depends on how your ranking metric is defined.

Reading the Enrichment Plot

The enrichment plot has three panels:

  1. Top panel (ES curve): The green line shows the running enrichment score. A clear peak away from zero = strong enrichment. The stats box shows NES, FDR, p-value, and gene set size.
  2. Middle panel (hit markers): Black vertical bars mark where each gene in the gene set falls in the ranked list. Clustering of bars on one side = enrichment.
  3. Bottom panel (ranking metric): The waterfall plot showing each gene's ranking value. Red = positive (upregulated/enriched), blue = negative (downregulated/depleted).

FDR Interpretation

> 0.25 Not significant. The gene set is not enriched beyond what's expected by chance.
