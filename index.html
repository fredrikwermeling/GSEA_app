How to use Enrich

Enrich runs Gene Set Enrichment Analysis (GSEA) entirely in your browser. No data is uploaded to any server.

1. Prepare your data

Your input file should be a .csv or .tsv with at least two columns:

  • Gene name column — standard gene symbols (e.g. TP53, BRCA1)
  • Ranking metric column — a numeric value to rank genes (e.g. log2 fold change, t-statistic, or signal-to-noise ratio)

Save your spreadsheet as CSV (comma-separated) or TSV (tab-separated) from Excel, Google Sheets, or R. Ensure the first row contains column headers.

2. Upload and configure

  1. Click Choose file or use Load Example Data to try with sample data
  2. Select which column contains gene names and which contains the ranking metric
  3. Genes will be automatically sorted from highest to lowest metric value

3. Choose gene set collections

Select one or more MSigDB collections (Hallmark is recommended to start). You can also upload a custom GMT file.

4. Run the analysis

Click Run GSEA. The analysis runs in your browser using a Web Worker. With Hallmark (50 sets) this takes seconds; larger collections take longer.

5. Interpret the results

  • Overview — Lollipop plot showing top enriched gene sets sorted by NES
  • Enrichment Plot — Classic GSEA enrichment score curve for individual gene sets
  • Overlap — Jaccard similarity heatmap of significant gene sets
  • Results Table — Full sortable/filterable table with CSV export

Key statistics

  • NES (Normalized Enrichment Score) — positive = upregulated, negative = downregulated
  • FDR (False Discovery Rate) — <0.25 is suggestive, <0.05 is significant
  • Leading Edge — the subset of genes driving the enrichment signal

Enrichment Score Plot iHow to read this plot:
Top panel — The green running enrichment score (ES) curve. A peak away from zero indicates enrichment. Peaks to the left = genes are upregulated; peaks to the right = downregulated.
Middle panel — Black bars show where genes in this set appear in the ranked list.
Bottom panel — The ranking metric for all genes (red = positive/upregulated, blue = negative/downregulated).
Leading edge = genes before the ES peak that drive the signal.
Gene Set Info i NES — Normalized Enrichment Score. Positive = genes tend to be upregulated; Negative = downregulated.
FDR — False Discovery Rate (Benjamini-Hochberg adjusted).
p-value — Nominal enrichment p-value from permutation test.
ES — Raw (unnormalized) enrichment score.
Leading Edge — Genes appearing before the ES peak; these drive the enrichment signal.
Direction — Whether the gene set is enriched among upregulated (positive metric) or downregulated (negative metric) genes.
Status — ★ Significant (FDR < 0.05), ◆ Suggestive (FDR < 0.25), or ○ Not significant.

Select a gene set to see details.

Search Genes iEnter gene names to find their position in the ranked list. Genes are highlighted with labeled orange lines on the enrichment plot. You can paste a list from Excel (one gene per line) or use commas to separate names.
Top & Bottom Genes iShows the genes with the highest (top) and lowest (bottom) ranking metric values in your dataset. This helps orient the analysis: top genes are the most upregulated, bottom genes are the most downregulated.

Run GSEA to see top/bottom genes.

Leading Edge Genes iAll genes belonging to this gene set, shown in ranked order.
Rank — Position in the full ranked gene list.
Gene — Gene symbol.
Metric — The ranking metric value (e.g. log2FC).
Running ES — Cumulative enrichment score at this gene's position.
Leading Edge — "Yes" if the gene appears before the ES peak and contributes to the enrichment signal.
